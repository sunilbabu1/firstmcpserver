# Installation Steps

To install the MCP Server, run the following command:

'''json
{
"mcp-server": {
      "command": "uvx",
      "args": [
        "--from",
        "git+https://github.com/sunilbabu1/firstmcpserver.git",
        "mcp-deploy"
      ]
    }
}
'''

This will fetch and set up the mcp-server from the specified Github repository.
